National Walkability Index

EPA National Walkability Index – Nationwide

What is this project?

This app visualizes the EPA's National Walkability Index for all U.S. states using block‑group data from the Smart Location Database.

The index is built at the census block group level and combines density, land-use mix, street connectivity, and transit access into a single walkability score.
